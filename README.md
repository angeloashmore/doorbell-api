## Instalation

* Clone project
* Run `bundle install`
* Copy `config/mongoid.yml.sample` to `config/mongoid.yml`
* Run `rackup`

## Specs

* Run `[bundle exec] rspec`

## Routes

* Run `[bundle exec] rake routes`

## Database

* [Mongodb](https://www.mongodb.org/) using [Mongoid](http://mongoid.org/)

## API

* [Grape](https://github.com/intridea/grape)
